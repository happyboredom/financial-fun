Installation
----

Grab from github and install npm packages.

```
git clone git@github.com:happyboredom/financial-fun
cd financial-fun
npm install
```

Run dev server
```
npm run start
```

Package up for deploy
```
npm run build
> look in ./build dir
```

Open web browser http://localhost:3000
