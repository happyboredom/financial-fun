Grab from github and install npm packages.
----

```
git clone git@github.com:happyboredom/financial-fun
cd financial-fun
npm install
```

Run dev server
```
npm run start
```
Open in browser http://localhost:3000


Run test suite
----
```
npm run test
```

Package up for deploy
----
```
npm run build
> look in ./build dir
```
